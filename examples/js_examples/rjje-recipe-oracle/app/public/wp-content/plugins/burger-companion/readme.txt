== Description ==

This plugin creates the additional sections on the home page in burger themes. enjoy the full functionality of the Burger Software by installing the Burger Companion plugin.
Burger Companion is a plugin created to enhance the functionality of the WordPress theme created by burgersoftware.

== Features ==

Slider Section
Info Section
Service Section
Call To Action Section 
And many more.
